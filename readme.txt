== Description ==

### PPM FAQ by http://perfectpointmarketing.com

This plugin will add an expand collapse FAQ feature inside a post or page.

Plugin Features

* Shortcode System
* Wordpress Custom Post Enabled. 
* TinyMCE Button added for generating Shortcode.
* Very Lightweight. Only 2KB.
& many More.

Live Preview: http://perfectpointmarketing.com/plugins/ppm-faq

== Installation ==

1. Install as a regular WordPress plugin
2. The plugin supports custom post. After installing you can see a menu called FAQ. There, you can add faqs. After adding FAQ you can use shourcode `[ppm_faq]`
